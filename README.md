# Spotiup

## About

This is Python script for Pomodoro method.

It creates Spotify playlist about 30min.

```sh
python app.py # random genre and random tempo
python app.py --tempo 120 # random genre and tempo 120
python app.py --genre chill # genre chill and tempo random
python app.py --tempo 80 --genre chill
```

## Installation

First, Should to create Spotify Developer App.

To get Client ID, Client Secret, and Redirect URI.

```envrc .envrc
export SPOTIPY_CLIENT_ID='123456'
export SPOTIPY_CLIENT_SECRET='123456'
export SPOTIPY_REDIRECT_URI='http://127.0.0.1:9090'
```

```sh
python -m venv venv
. ./venv/bin/activate
pip install spotipy
python app.py --tempo 80 --genre chill
```
